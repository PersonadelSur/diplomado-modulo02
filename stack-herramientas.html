● Chats IA Generativa
Claude Free + Pro
Anthropic · Razonamiento largo, archivos, análisis
Discover Define Develop
  • Sintetizar entrevistas de usuario en insights accionables
  • Analizar documentos largos (research, reportes, transcripciones)
  • Generar arquetipos y problem statements estructurados
  • Construir frameworks de priorización (RICE, MoSCoW)
 Ideal para trabajar con contexto largo — sube tus entrevistas y pide síntesis por patrones de comportamiento
ChatGPT Free + Plus
OpenAI · GPT-4o, análisis, generación
Discover Define
  • Generar hipótesis de usuario desde cero
  • Comparar frameworks de producto y metodologías
  • Crear estructuras de entrevistas semi-estructuradas
 GPT-4o con visión permite analizar wireframes y capturas de pantalla para feedback de UX
Gemini Free + Advanced
Google · Integración con Workspace, multimedia
Discover Define
  • Analizar datos de Google Sheets directamente
  • Integrar con Gmail y Docs para síntesis de conversaciones
  • Gemini Gems: agentes especializados por rol de investigación
 Los Gems permiten crear un "investigador de usuario" especializado que mantiene contexto de tu producto
Grok Free + X Premium
xAI · Acceso a datos en tiempo real de X/Twitter
Discover
  • Detectar tendencias emergentes de usuarios en redes sociales
  • Analizar conversaciones reales sobre tu categoría de producto
  • Benchmark de competidores desde señales sociales
 Único en acceso a datos de X en tiempo real — útil para detectar dolor de usuarios que expresan frustración pública
● Investigación Profunda
Perplexity Free + Pro
Búsqueda con IA · Fuentes citadas, actualizado
Discover
  • Benchmark de competidores con fuentes verificables
  • Investigar tendencias de industria con información actualizada
  • Contexto de mercado antes de iniciar discovery
 A diferencia de otros chats, cita fuentes — fundamental cuando necesitas evidencia para sustentar decisiones ante stakeholders
NotebookLM Gratuito
Google · Base de conocimiento con tus propios archivos
Discover Define
  • Subir todas las entrevistas y extraer patrones comunes
  • Crear una base de conocimiento del usuario que persiste
  • Hacer preguntas cruzadas entre múltiples documentos de research
  • Generar podcasts de audio de tus insights para el equipo
 El PM lo usa como "cerebro del research" — consolida todo el conocimiento cualitativo del equipo en un solo lugar consultable
● Transcripción y Análisis de Entrevistas
Granola Free + Pro
Transcripción IA de reuniones con síntesis automática
Discover
  • Transcribir entrevistas de usuario automáticamente
  • Generar resúmenes con highlights y citas clave
  • Exportar transcripciones para análisis en NotebookLM o Claude
 Elimina la carga de transcripción manual — el PM puede focalizarse en escuchar y preguntar durante la entrevista
● Diseño Visual y Presentación
Canva IA Free + Pro
Diseño con IA generativa integrada · Magic Design
Define Develop
  • Crear journey maps y arquetipos visualmente
  • Diseñar entregables de research para presentar a stakeholders
  • Generar imágenes para personas y escenarios de usuario
 Convierte tus insights en materiales visuales sin necesidad de un diseñador — ideal para presentaciones internas rápidas
Gamma Free + Pro
Presentaciones generadas con IA desde texto o prompt
Define Deliver
  • Crear presentaciones de hallazgos de research en minutos
  • Documentar el problem statement para alineación del equipo
  • Convertir un documento de insights en deck ejecutivo
 El PM pega su síntesis de research y genera una presentación lista para stakeholders — reduce tiempo de comunicación dramáticamente
● Prototipado Rápido (Referencia Módulos 3–6)
Bolt Free + Pro
Generación de apps web desde texto · StackBlitz
Develop
  • Prototipo funcional de una hipótesis de solución en horas
  • Generar UI para validar flows antes de involucrar al equipo de dev
 Permite al PM crear un MVP clicable para probar con usuarios sin depender del sprint — acorta el ciclo de validación
Lovable Free + Pro
Generación de apps React desde conversación
Develop Deliver
  • Crear prototipos de alta fidelidad funcionales para test de usabilidad
  • Iterar visualmente el producto en conversación con el equipo
 Más potente que Bolt para productos complejos — útil cuando el prototipo necesita lógica de negocio real
● Automatización y Flujos de Trabajo (Referencia Módulos 5–6)
n8n Free + Cloud
Automatización open source · Flujos visuales con IA
Deliver
  • Automatizar recopilación de feedback de usuarios
  • Conectar herramientas del stack de producto sin código
  • Flujos de síntesis: entrevista → transcripción → resumen → Notion
 El PM orquesta el pipeline de datos de usuario sin depender de engineering — reduce tiempo de ciclo de aprendizaje
Make Free + Pro
Automatización visual · Integración con 1000+ apps
Deliver
  • Conectar formularios de research con Notion o Sheets automáticamente
  • Automatizar notificaciones de métricas de producto al equipo
 Más accesible que n8n para equipos sin perfil técnico — buena entrada a automatización de workflows de discovery
Google Workspace Studio Gratuito
IA integrada en Google Sheets, Docs, Slides · Gemini
Define Deliver
  • Analizar datos de usuario directamente en Sheets con IA
  • Generar resúmenes de documentos de research en Docs
  • Crear dashboards de métricas con asistencia de IA
 Si el equipo ya usa Google Workspace, es el punto de entrada más rápido a IA sin cambiar de herramientas
